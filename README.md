# Onliner Forum Dark + Filter

Расширение для браузера для `forum.onliner.by`.

## Возможности

- Применяет тёмную тему через `themes/dark-base.css`. Загрузка стилей реализована через css в manifest.json, чтобы избежать возможного промаргивания родной темы форума.
- Скрывает изображения внутри спойлеров форума. Сокрытие для цитат не реализовано, так как хочется верить в адекватность цитирующих пользователей.
- Поддерживает правила фильтрации по отдельным пользователям. При этом скрывается не только текст пользователя, но и колонка автора, так как если бы он не писал сообщения вовсе.

## Установка

1. Откройте страницу управления расширениями в браузере Google Chrome.
2. Включите режим разработчика.
3. Загрузите эту папку как распакованное расширение.

## Примечания
- Настройки пользователей сохраняются в `chrome.storage.sync`.
- Расширение работает для `https://forum.onliner.by/*`.
